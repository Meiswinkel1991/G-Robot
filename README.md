# GMX Grid Trading Bot

This bot tries to execute a grid trading strategy on the protocol GMX.

Try running some of the following tasks:

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat run scripts/deploy.js
```
